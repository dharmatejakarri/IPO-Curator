What you're building

An AI-powered, RAG-based IPO analysis platform that ingests historical + upcoming IPO data from sources like Groww, NSE, and BSE — and delivers unbiased, AI-generated summaries, risk scores, and smart nudges for each IPO. Zero cost. Fully deployed. End-to-end.

Core features (from your notes)

  • 1
    RAG-based IPO Analysis EngineFeed IPO prospectuses, company data, GMP data, financial context into a vector DB. LLM retrieves and reasons over it to generate summaries.
  • 2
    Agentic Suggestions (Nudges)An AI agent that cross-references current macro context, sector trends, and past IPO performance to produce actionable IPO recommendations.
  • 3
    Historical + Upcoming IPO DatabaseData pipeline that auto-scrapes/fetches IPO data from Groww, NSE, IPO Alerts API, and stores it in Supabase/Neon.
  • 4
    User Registration + Lead CaptureSimple email sign-up, saved watchlist, newsletter nudges — stored in your free-tier PostgreSQL database.
  • 5
    Live Web App (Public URL)Fully deployed, shareable link to show recruiters. Clean, professional UI.

Recommended zero-cost tech stack

Every tool below has a genuinely free tier suitable for a portfolio project. No credit card traps.

Frontend

Next.js 14 (App Router)Top pick
React-based, full-stack, SSR + API routes in one repo. Vercel deploys it for free with zero config. This is the #1 framework recruiters recognize.
Tailwind CSS + shadcn/uiFree
Production-quality UI with minimal effort. shadcn components are copy-paste, zero dependencies. Looks impressive in demos.

Backend / API

FastAPI (Python)Top pick
The de-facto standard for AI/ML backends. Async, typed, auto-generates Swagger docs. Recruiters in the AI/ML space expect this. Run on Render free tier.
LangChain / LangGraphFree
Industry-standard RAG + agentic framework. LangGraph adds the "agentic" layer — multi-step reasoning, tool calling. This alone impresses interviewers.

LLM (free tier)

Google Gemini 2.5 FlashPrimary
1,000 requests/day free. 1M token context window. No credit card needed via AI Studio. Fastest free LLM for RAG analysis.
Groq (Llama 3.3 70B)Fallback
300+ tokens/sec. Blazing fast. Use for real-time analysis views where speed matters. Free tier is generous for demo traffic.

Vector Database (RAG)

Chroma DB (local/self-hosted)Top pick
100% free, open-source, runs in-process with Python. Zero infra setup. Perfect for portfolio — and you can swap to Pinecone/Qdrant later.
Pinecone (free tier)Alternative
Managed, cloud-hosted. 1 free index, 100K vectors. Good for showing "production-grade" tooling to recruiters. Start here if you want zero ops.

Database (user data + IPO store)

SupabaseTop pick
Free PostgreSQL + auth + REST API + realtime. Open-source Firebase alternative. 500MB storage, 2 projects free. Perfect for user capture + IPO data table.
Neon (serverless Postgres)Alternative
Serverless Postgres, scales to zero (no idle charges). 0.5 GB storage free. Great if you're already using Render for the backend.

Embeddings (free)

sentence-transformers (HuggingFace)Top pick
all-MiniLM-L6-v2 runs locally, zero API cost, fast. Generate embeddings for IPO documents yourself. No rate limits ever.
Gemini Embedding APIAlternative
text-embedding-004, 1500 RPD free. Slightly better quality than MiniLM. Use if you prefer API-based over local model.

Full SDLC — realistic 8-week build plan

Each phase is self-contained. You can pause after any phase and still have something demo-able.

Week 1
Requirements + Design
Define user stories, sketch UI wireframes (Excalidraw), set up GitHub repo, create Supabase project, list all IPO data fields needed.
Week 2
Data Pipeline
Write Python scrapers for NSE/ipoalerts.in/Groww. Store to Supabase. Download 20-30 IPO PDFs (DRHP). Schedule daily cron with APScheduler.
Week 3
RAG Core (Backend)
FastAPI setup. LangChain document loader for PDFs + structured data. MiniLM embeddings. ChromaDB vector store. Basic Q&A endpoint working locally.
Week 4
LLM Integration + Prompting
Connect Gemini Flash. Write system prompts for IPO summary, risk analysis, subscription recommendation. Test output quality. Add Groq fallback.
Week 5
Agentic Layer (LangGraph)
Build multi-step agent: fetch macro context → retrieve sector comps → cross-reference GMP → generate nudge. This is the "agentic framework" from your notes.
Week 6
Frontend (Next.js)
Build IPO listing page, individual IPO analysis page, AI chat interface, user sign-up (Supabase Auth). Tailwind + shadcn for polish.
Week 7
Deployment + Integration
Deploy FastAPI to Render. Deploy Next.js to Vercel. Connect env variables. Test full flow end-to-end. Fix cold start issues on Render free tier.
Week 8
Polish + Recruiter Packaging
Add README with architecture diagram. Record Loom demo video. Write a 500-word Medium/dev.to article about the project. Add to GitHub portfolio.

Deployment — what actually works at zero cost

Key reality: Render free tier sleeps after 15 min of inactivity (cold start ~30s). Use UptimeRobot (free) to ping it every 10 min and keep it warm during demo hours.
